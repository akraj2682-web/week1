# Personal Introduction Python Project

## Project Overview
This is a simple Python program that collects user information such as name, age, and hobby.  
It then displays a friendly welcome message using the provided information.

## Objectives
- Learn how to use input() in Python
- Store user information using variables
- Display output using print()
- Use f-strings for formatted messages
- Write clean and understandable code with comments

## How to Run the Program

1. Install Python on your computer
2. Download or clone this repository
3. Run the following command:

python personal_intro.py
