# Personal Introduction Python Program

## Project Overview

This project is a simple Python program that collects basic information from a user such as their **name, age, and hobby**. After collecting the information, the program displays a **friendly and personalized welcome message**.

The program demonstrates fundamental Python concepts such as user input, variables, formatted output, and writing readable code with comments.

---

## Project Objectives

* Learn how to use the `input()` function to collect user information.
* Store user responses using variables.
* Display output using the `print()` function.
* Use **f-strings** for formatted and readable output.
* Write clean code with comments for better understanding.
